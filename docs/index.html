Download Bloo

Getting Started

Step One

First, you need to unzip the downloaded zip file into an easy to find directory on your local development machine. Please note that no live server is required to run Bloo!

As you can see from the image below, I chose to create a directory titled websites and stored the unzipped folder inside this folder. Where you store the file is up to you, just be sure you can find it easily.

Step Two

Second, you must run firebase init in your terminal. This will initialize Firebase into the project directory. The majority of the process is simply default however, there are a few exceptions and they are outlined below.

Which Firebase CLI features do you want to setup for this folder? You will need to select the following:

  • Database
  • Functions
  • Hosting
  • Storage
What do you want to use as your public directory?
  • Type app into your terminal and then press Enter.
Configure as a single-page app?
  • You will want to type N for no. We use our own routing system.

Once you have completed initialization of firebase, Bloo is ready to begin running. If you've never used Firebase before, you can follow their getting started guide on their website.
